# Project Ascension: Gamification Features

## Overview
The app has been transformed into a "Learning Game" with the following features:

### 1. World Map Dashboard
- **Path-Based Progression**: Navigate through 8 distinct modules/worlds.
- **Locking System**: You must earn at least **1 Star** (30% score) in a module to unlock the next one.
- **Visuals**: Pulsing nodes, user avatar marker, and animated dashed path.

### 2. Gamification Engine
- **XP & Levels**: Earn XP to level up. Levels are calculated logarithmically (harder to level up as you go).
- **Streaks**: Daily activity tracking. Don't break the chain!
- **Cloud Sync**: Your XP, Level, and Streak are synced to your Firebase account.

### 3. Challenge Mode (The Game)
- **Combo System**: Answer consecutive questions correctly to build a Combo multiplier.
- **Speed Bonus**: Answer in under 5 seconds for +5 XP.
- **XP Popups**: Instant visual feedback on earnings.
- **Result Screen**: See your Stars, XP breakdown, and Level progress at the end of each session.

## How to Play
1. Click the first unlocked node (Core Java).
2. Complete the 10-question challenge.
3. Earn XP and Stars to unlock the next node.
